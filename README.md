## üì• Installation

### Plugin Installation
1. Download the latest `Revive-Players-GoldKingZ.x.x.x.zip` release
2. Extract contents to your `csgo` directory
3. Configure settings in `Revive-Players-GoldKingZ/config/config.json`
4. Restart your server
